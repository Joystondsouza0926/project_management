# Product Opportunity Intake App

A Streamlit application for managing product opportunities, including intake, review, and decision tracking.

## Setup

1. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```

2. Run the app:
   ```bash
   streamlit run app.py
   ```

## Features
- **Summary View:** List all opportunities with status and priority.
- **Detail View:** Drill down to edit details, make decisions (Accept/Reject/Park), and export to Excel.
- **Data Persistence:** Saves data to a local CSV file.
